MSVC 6.0 required!

1) Install freeimage241 and fmodapi375win into same dir as blitzplus.

2) Build bblaunch, blitzcc, blitzplus, debugger, ide, linker projects in release config.

3) Build blitzplus project in debug config.

4) Output files in blitzplus/_release dir.

5) Copy fmodapi375win/api/fmod.dll to blitzplus/_release/bin.

6) Go!
